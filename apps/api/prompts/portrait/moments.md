# 想象用户的朋友圈

你拿到的是一个用户最近的几段对话。基于此**虚构** ta 可能在朋友圈里发的内容。**这是练习共情、同时把"AI 眼中的 ta"翻译成可视画面**——不是要骗用户，所以不必力求精准、可以带点你的猜测和臆测。

## 几条 vibe

- 写 5 到 8 条朋友圈，跨度从最近 1 周到最近 1 个月
- 每条带：发布相对时间、正文（短，符合朋友圈节奏）、可选评论 0-3 条（若有评论，author 用 ta 的可能朋友的化名："小 K"、"老周"、"H" 这种）
- 题材覆盖：**至少要有不同主题**——纯吐槽、生活流水、深夜想法、爱好/兴趣、跟工作有关、分享（图/链接）等等。**不要全都写在 ta 跟 bot 聊过的同一主题上**——朋友圈是 ta 没在 AI 这里说的部分。
- 偶尔留一条**很可能并非真实** but plausibly characteristic 的 — 朋友圈本来就有点表演成分。

## 含图

如果 `with_image=true`：每条朋友圈给一段 `image_prompt`（中文一句话，描述 ta 可能配的图）。否则置 null。

## 输出

严格输出一个 JSON 对象：

```json
{
  "items": [
    {
      "posted_relative": "3 小时前",
      "text": "正文，自然口语，不要正经",
      "image_prompt": "或 null",
      "comments": [
        { "author": "小 K", "text": "评论" }
      ],
      "ai_note": "你为什么想象 ta 会发这条 — 一句话给用户看"
    }
  ]
}
```

只输出 JSON 对象，不要写任何前后说明、不要 wrap 在 markdown 代码块外面（可以 wrap 在 ```json ... ``` 里）。
