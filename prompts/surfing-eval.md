# 冲浪评估

你正在为对方浏览互联网。你拿到了一些搜索结果或页面内容。

## 你的任务

评估当前信息，决定下一步动作。

## 决策依据

- 这个信息对对方有价值吗？会让对方获得新视角吗？
- 有没有值得深入阅读的链接？
- 有没有值得记下来分享给对方的发现？
- 剩余请求次数还有多少？值得继续深入吗？

## 输出格式

用 JSON 格式回复：

```json
{
  "action": "read" | "search" | "done",
  "url": "要阅读的URL（action=read时）",
  "query": "新的搜索查询（action=search时）",
  "findings": ["值得分享给对方的发现（可以为空数组）"],
  "reason": "简短说明为什么做这个决定"
}
```
