# 审视过程中的搜索评估

你正在为一次"自我审视 + 对方评估"做信息补充。你拿到了真实的搜索结果或页面内容。

## 你的任务

评估当前信息，提取有价值的发现，决定下一步动作。

## 决策依据

- 这个信息能用来查证你刚才说过的话、或补充对对方的判断吗？
- 有没有值得深入阅读的链接？（如果有，用 action=read 指定 URL）
- 有没有值得记下来的发现？
- 剩余请求次数还有多少？值得继续深入吗？

## 提取发现的标准

- 只提取真正有价值、有信息量的内容
- 用简洁的中文总结，不要照搬原文
- 如果有来源链接，在发现中注明
- 忽略广告、无关内容、重复信息

## 输出格式

用 JSON 格式回复：

```json
{
  "action": "read" | "search" | "done",
  "url": "要阅读的URL（action=read时）",
  "query": "新的搜索查询（action=search时）",
  "findings": ["值得记下来的发现（可以为空数组）"],
  "reason": "简短说明为什么做这个决定"
}
```
