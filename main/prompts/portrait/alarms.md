# 想象用户的闹钟

基于用户最近的对话，**虚构** ta 手机里可能设的闹钟。一个人的闹钟列表能透露很多 ta 没明说的东西——作息、健康、约束、自我管理风格。

## 几条 vibe

- 6 到 10 条
- 每条：时间（HH:MM 24 小时制）、label（短）、repeat（"每天"/"工作日"/"周一三五"/"一次性"）、enabled（true/false——显然有些是设了又懒得删的）
- 类型混合：**起床 / 提醒喝水 / 服药 / 出门 / 通勤切换 / 吃饭 / 哄娃 / 健身 / 半夜深度工作**
- 把那种**"两个相隔 5 分钟的起床闹钟"这种用户自我博弈**的细节也加上——更真实
- 一些 enabled=false 的 — 闹钟是会留旧痕迹的

## 含图

`with_image=true` 时不一定要给图（闹钟不太需要配图）；如要给，可以给"闹钟铃声主题"或表盘界面描述。多数 null 即可。

## 输出

```json
{
  "items": [
    {
      "time": "07:25",
      "label": "起床（再五分钟）",
      "repeat": "工作日",
      "enabled": true,
      "image_prompt": "或 null",
      "ai_note": "为什么觉得 ta 设这个 — 一句话"
    }
  ]
}
```

只输出 JSON 对象。
