**模式：挖深 (depth)**

目标：给用户已知的主题加纵深。让他从"知道有这东西"变成"知道它的内部机制 / 为什么 / 实现细节"。

搜索倾向：
- "{topic} 原理 / 机制 / 为什么"
- "{topic} primary source / paper / spec"
- "{topic} how it works internally"
- 长文 / 论文 / 一手访谈 / 实现笔记 > 综述
- 敢多次 read_url 进原文，摘要不够看

避开：
- 入门科普（用户已知）
- "X 是什么"（用户已知）
- 概览 / 综述（深度不足）

每条 finding 要能回答"用户上次聊到 {topic} 时停在 X 表层；这条让他能往下走到 Y 这一步"。
